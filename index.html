Welcome to my WASM Tutorial!

This is a simple tutorial on WebAssembly!

Requirements

First of, you need at least a github account, and if you have Python in your device, it needs to be Python 3.6 or newer.
This tutorial will focus on C, so make sure you can compile C on your device!
Once that's out of the way, follow the next steps according to your Operating System.

Linux

Make sure you have the latest Python3 version installed, as well as git.
You alse may install Cmake, although it's not necessary.

MacOS

Install the XCode Command Line Tools, which will allow to install git, if git is already install you may skip this step.
Install Cmake.

Windows

Make sure you have Python 3.6, or a newer release, installed.

Getting the Emscripten SDK

Emscripten is the tool we are going to use to generate our .wasm file from our C code.

You can check the official Emscripten downloading tutorial here, altough in this tutorial we'll cover al necessary steps.

You'll need to open your device's terminal, and once you find where you want to download Emscripten, you'll need to type git clone https://github.com/emscripten-core/emsdk.git on your console.
Once its done type "cd emsdk" and then "./emsdk install latest" to install Emscripten.
Finally, you'll need to type "./emsdk activate latest" to activate Emscripten.

Using the Emscripten SDK

Once you activate Emscripten, you need to type "source ./emsdk_env.sh" to in the /emsdk directory to enable it in your terminal. (You need to repeat this everytime you wish to use Emscripten).

IMPORTANT!

  1. On Windows, run "emsdk" instead of "./emsdk", and "emsdk_env.bat" instead of "source ./emsdk_env.sh".
  2. On Windows, if you use the activate command, the step of emsdk_env.bat is optional.

Once that's done, all you need to do is, on your current terminal, navigate to the directory where you have your .c file you wish to turn into wasm, and type:
"./emcc filename.c -o newname.html"
This will not only generate the .wasm file containing your C code, but also generate a .html file running said wasm code.

You are Done!

You can now run your .html file in your browser, or use your .wasm file in other projects.

Have fun!
